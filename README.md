# my little homebrew tap

A [Homebrew](https://brew.sh) tap for tools by [@sassman](https://github.com/sassman).

## Formulae

- [`amoxide`](https://github.com/sassman/amoxide-rs) — Shell alias manager; manage aliases globally via profiles or per-project
- [`amoxide-tui`](https://github.com/sassman/amoxide-rs) — Interactive TUI for amoxide
- [`ssd-benchmark`](https://github.com/sassman/ssd-benchmark-rs) — Super Simple Disk Benchmark; benchmarks the writing performance of your disk

## Casks

- [`display-align`](https://github.com/sassman/display-align) — Automatic display arrangement for macOS

## Usage

```sh
brew install sassman/tap/ssd-benchmark
brew install --cask sassman/tap/display-align
```
